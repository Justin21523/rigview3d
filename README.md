# RigView3D

RigView3D is a lightweight, front-end-only 3D viewer built with Vite, TypeScript, and Three.js.

## Learning notes
Beginner-friendly implementation notes (Chinese) live in `docs/implementation/README.md`.

## Quick start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Typecheck

```bash
npm run typecheck
```
